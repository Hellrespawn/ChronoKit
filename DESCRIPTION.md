Control game speed in Factorio. Speed up time to fast-forward through quiet periods, slow it down to handle complex situations. Automatically resets when taking damage. When a player-force entity takes damage while the game is running above normal speed, the game resets or pauses (configurable). A GPS link is printed to chat.

## Controls

| Button | Action |
|--------|--------|
| ⏸ / ▶ | Toggle pause |
| `<` | Halve game speed |
| `>` | Double game speed |
| `x1` / `xN` / `/N` | reset to ×1 or restore saved speed. |
