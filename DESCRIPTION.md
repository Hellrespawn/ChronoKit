Control game speed in Factorio. Speed up time to fast-forward through quiet periods, slow it down to handle complex situations. When a player-force entity takes damage while the game is running above normal speed, ChronoKit reacts automatically: do nothing, reset speed to ×1, or pause the game (configurable, cycle via button). A GPS link is printed to chat.

## Controls

| Button | Action |
|--------|--------|
| ▶ / ⏸ / `<` / `>` | Left-click: increase speed. Right-click: decrease speed. Shift-click: toggle pause. |
| `x1.0` / `xN.N` / `/N.N` | At ×1 or paused: restore saved speed. While fast-forwarding: reset to ×1. |
| • / 🔄 / 🚫 | Cycle the damage behaviour: do nothing → reset speed → pause game (right-click to reverse). |
